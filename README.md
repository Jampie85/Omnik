# Omnik
Omnik server settings chrome plugin (tested on Omnik2000tl)

Makes it possible to edit server A on Omnik inverter.


![alt text](https://github.com/Jampie85/Omnik/blob/main/omnikserver.png?raw=true)

INSTALL:
Go to chrome://extensions/ -> Enable Developer Mode -> Load unpacked extension -> select folder with the files.
